Pyternary

There is a Computation Engine called Deternary, which is a really impressive program. However, it lacks the ability to save your code, and also is a lot more confusing to enter your code. Because of these issues, I created Pyternary, which is a Python based interpretor that allows you to program outside of the computation engine. It will then import the code for you. This entire project took roughly 3-6 hours.
